RSA Key Generator

1. Controls

Click "Generate keys" to create a new RSA keypair. Click "Reset" to clear the fields.

2. Generated keys

Public key (e, n)

The public key is used to encrypt data. You can share it with others.


Private key (d, n)

The private key is used to decrypt data. Keep it secret.


3. What are e, d, and n?

  • n: the modulus, equal to the product of two large primes. It is shared between the public and private keys.
  • e: the public exponent, used together with n to encrypt data or verify signatures. This value is public.
  • d: the private exponent, used together with n to decrypt data or create signatures. This value must be kept secret.
